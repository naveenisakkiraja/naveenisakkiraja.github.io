International Standard Atmosphere Calculator


SI US










Description

Compute the basic properties of the standard atmosphere at a specified altitude up to 86 km from sea level using the U.S. Standard Atmosphere Model, 1976 published by the U.S. Government Printing Office, Washington, D.C.
The user may choose between SI or US units. It accepts altitude as an input argument and returns non-dimensional values of temperature, pressure, density and the output values has been validated against the NASA-TM-X-74335 report.
